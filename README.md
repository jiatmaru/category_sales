# Category Sales Analysis（適性検査カテゴリ 716 売上分析）

本プロジェクトは、適性検査カテゴリ（category_id = 716）を対象とした  
売上分析・重複排除・四半期推移の可視化などを行うための分析コード・データ構造説明をまとめたものです。

実データ(JSON / SQL / CSV など)は機密情報のため、リポジトリには含めていません。  
代わりに **データ形式のサンプル（Markdown）** と、**使用したデータセットの種類** を公開しています。

---

# 📊 分析内容の概要

このプロジェクトでは、適性検査カテゴリ（id=716）の売上データを中心に以下の分析を実施しました。

- 四半期売上推移の可視化（確定・未確定含む）
- 重複注文除外（dedup）処理の適用
- 四半期別、月別、商品別の売上集計
- 四半期確定値と速報値の差分分析
- 個票レベル（detailデータ）分析による要因深掘り
